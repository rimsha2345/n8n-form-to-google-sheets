# Form to Google Sheets Automation (n8n)

This workflow automatically saves form submissions into Google Sheets and sends an email notification after each submission.

## Workflow Overview
The automation is triggered when a user submits a form. The submitted data is stored in a Google Sheet, and a confirmation or notification email is sent automatically.

## Tools Used
- n8n
- Google Sheets
- Gmail
- Webhooks / Form Trigger

## How It Works
1. User submits a form  
2. Data is automatically added or updated in Google Sheets  
3. An email notification is sent using Gmail  

## Use Case
This workflow is useful for:
- Lead collection  
- Contact forms  
- Event registrations  
- Feedback forms  

## Result
- No manual data entry required  
- Faster response time  
- Centralized data storage  
- Automatic email confirmation
